# 收藏链接管家 · Bookmark Manager

> **知行合一版 v2.0** · 不只看，要干！先判断能否落地执行，再收藏到知识库

LobeHub Agent Skill for intelligent bookmark management with **action-first** philosophy.

## 核心理念

- ⚡ **先落地，后收藏** — 收到链接先判断内容能否直接执行
- 🗂️ **智能沉淀** — 结构化提取内容，自动写入 Obsidian
- 🔌 **设备感知** — 在线时写入本地 Vault，离线时暂存

## 支持平台

**国内**：今日头条 · 小红书 · B站 · 知乎 · 公众号 · 抖音 · 快手
**国际**：YouTube · X (Twitter)
**通用**：任何网页链接

## 安装

npx skills add zhilinYu/lobe-skills@bookmark-manager

## 使用

直接发送链接给 Agent，自动解析、执行、收藏。
