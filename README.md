# Bookmark Manager

LobeHub Agent Skill for smart bookmark management.

## Supported Platforms

Domestic: Douyin, Kuaishou, Xiaohongshu, Bilibili, Zhihu, Toutiao, WeChat Official
International: YouTube, X (Twitter)
General: Any web link

## Features

- Auto-detect link source platform
- Extract title, author, summary, key points
- Structured Markdown storage
- IMA knowledge base sync
- Batch processing
- Smart tag recommendations
## Usage

Send a link to the AI agent, it will automatically parse and save.
