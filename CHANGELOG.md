# Changelog

## [2.0.0] - 2026-06-19

### 知行合一版 - 全面升级

#### 新增
- 核心原则「先判断落地再收藏」：收到链接自动判断能否在本地执行，能办直接办
- 双流程设计：可落地->执行+收藏 / 不可落地->仅收藏
- Obsidian Vault 深度集成：按平台自动分类、YAML元数据、标准模板
- 设备感知模式：在线直接写入本地Vault，离线暂存云端自动同步
- 落地执行记录：YAML中标记 executed/executed_summary

#### 改进
- 完整8步操作指南
- 标签管理 & 去重检查
- 多平台来源识别增强
- HTTP 503 自动重试机制

## [1.1.0] - 2026-06-18

### 新增
- 抖音、快手平台支持
- YouTube 平台支持
- X (Twitter) 平台支持
- 视频类内容提取规范（时长/播放量/时间戳）

## [1.0.0] - 2026-06-17

### 初始版本
- 基础链接收藏功能
- 今日头条/小红书/B站/知乎/公众号 平台识别
- Agent Documents 结构化笔记沉淀
- 智能标签推荐
