# 收藏链接管家 Skill

## 概述

这是一个智能收藏链接管理工具，能够自动识别来源平台、解析内容并沉淀到本地文档或腾讯 IMA 知识库。

## 支持的来源平台

### 🇨🇳 国内平台
- 头条/今日头条
- 小红书
- B站/哔哩哔哩
- 知乎
- 微信公众号
- 抖音
- 快手

### 🌍 国际平台
- YouTube
- X (Twitter)

### 通用
- 其他网页链接

## 平台识别规则

| 平台 | 域名/链接特征 | 内容类型 |
|------|--------------|----------|
| 抖音 | `douyin.com`, `v.douyin.com` | 短视频、图文 |
| 快手 | `kuaishou.com`, `v.kuaishou.com` | 短视频、直播 |
| YouTube | `youtube.com`, `youtu.be` | 视频、短片 |
| X/Twitter | `x.com`, `twitter.com` | 推文、帖子、线程 |
| 小红书 | `xiaohongshu.com`, `xhslink.com` | 图文笔记、视频 |
| B站 | `bilibili.com`, `b23.tv` | 视频、专栏、动态 |
| 知乎 | `zhihu.com`, `zhuanlan.zhihu.com` | 问答、文章 |
| 头条 | `toutiao.com` | 文章、视频 |
| 公众号 | `mp.weixin.qq.com` | 文章 |

## 工作流程

### 1. 链接解析流程

当用户发送收藏链接时：

1. **识别来源**：自动检测链接所属平台（支持短链接还原）
2. **内容抓取**：使用 Web Browsing 工具获取页面内容
3. **信息提取**：
   - 标题
   - 作者/来源
   - 发布日期
   - 核心摘要（2-3句话）
   - 关键要点（3-5个）
   - 行动项/启发（如有）
   - **视频类平台**：额外提取视频时长、播放量、关键时间戳
   - **社交媒体**：额外提取点赞数、转发数、热门评论
4. **结构化存储**：按标准格式保存

### 2. 存储目标

#### 本地存储（默认）
- 使用 Agent Documents 创建结构化 Markdown 笔记
- 按日期自动归类
- 支持标签分类

#### IMA 知识库同步（可选）
当用户要求「存入 IMA」或「保存到知识库」时：
1. 调用本地 IMA MCP 服务
2. 将内容同步到腾讯 IMA 知识库

## 使用方式

### 方式一：直接发送链接
```
用户：https://www.youtube.com/watch?v=xxx
AI：自动解析并保存，返回摘要
```

### 方式二：指定保存位置
```
用户：把这个抖音视频保存到 IMA：https://v.douyin.com/xxx
AI：解析后同步到 IMA 知识库
```

### 方式三：批量处理
```
用户：帮我整理这些收藏链接：
- https://youtube.com/watch?v=xxx
- https://x.com/user/status/123
- https://v.kuaishou.com/xxx
AI：逐个解析并统一保存
```

### 方式四：按平台筛选
```
用户：看看我收藏的 YouTube 视频
AI：筛选并展示所有 YouTube 相关收藏
```

## IMA MCP 配置

### 前置要求
1. 安装 IMA MCP 服务
2. 配置 Cookie 认证

### 服务管理命令

```bash
# 启动 MCP 服务
cd /path/to/tencent_ima_mcp && uv run python ima_mcp_server.py

# 直接保存链接到 IMA
cd /path/to/tencent_ima_mcp && uv run python ima_add.py <URL> \
  --title "标题" \
  --content "内容" \
  --tags "标签1,标签2"

# 测试连接
cd /path/to/tencent_ima_mcp && uv run python test_connection.py
```

## 高级功能

### 1. 智能标签推荐

根据内容自动推荐相关标签，如：

- 技术类：#前端 #后端 #AI #编程
- 生活类：#美食 #旅行 #健康
- 学习类：#教程 #课程 #书籍
- 视频类：#教程 #vlog #开箱 #评测
- 社交类：#热点 #观点 #讨论

### 2. 内容去重

检测重复链接，避免重复收藏。

### 3. 定期回顾

支持按时间、标签、平台筛选收藏内容，便于回顾。

### 4. 跨平台搜索

```
用户：搜索关于 "AI绘画" 的收藏
AI：在所有平台中搜索相关内容并展示
```

### 5. 收藏统计

```
用户：我的收藏统计
AI：展示各平台收藏数量、标签分布等
```

## 注意事项

1. **隐私保护**：不保存需要登录才能查看的私密内容
2. **版权尊重**：仅保存摘要和要点，不完整转载
3. **存储限制**：定期清理过期或无用的收藏
4. **视频内容**：依赖平台公开信息，无法提取付费/会员内容
5. **社交媒体**：X/Twitter 可能需要登录才能查看完整内容
